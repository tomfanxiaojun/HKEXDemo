# Front-End files for HKEX


## Quick Start

Several quick start options are available

* Clone the repo: `git@gitcn.epam.com:hkex-web1.git`
* Checkout the develop branch
* Run `npm install`
* Running `gulp`, the default task will open the browser with address `localhost:9000`


## Folder structure
[Structure of the folder, is based of Sass Guidelines Architecture](http://sass-guidelin.es/#the-7-1-pattern)

```

|
|– abstracts/
|   |– _variables.scss    # Sass Variables
|   |– _functions.scss    # Sass Functions

|– base/
|   |– _reset.scss        # Reset/normalize
|   |– _typography.scss   # Typography rules
|   …                     # Etc.
|
|– components/

|   …                     # Etc.
|

```
